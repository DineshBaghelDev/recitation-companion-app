Setup

1. Install Dependencies

cd BACKEND
pip install -r requirements.txt

2. Configure Environment

Copy .env.example to .env:

copy .env.example .env

Default configuration works out of the box. Modify .env if needed.

3. Run the Server

python -m app.main

Or using uvicorn directly:

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

API Endpoints

Base URL: http://localhost:8000

Health Check

  • GET / - API info
  • GET /health - Health status

Verses

  • GET /api/v1/slok/{chapter}/{verse} - Get specific verse
  • GET /api/v1/slok/{chapter} - Get random verse from chapter
  • GET /api/v1/chapters - Get all chapters
  • GET /api/v1/chapter/{chapter} - Get chapter details
  • GET /api/v1/chapter/{chapter}?include_verses=true - Get chapter with all verses

API Documentation

Once the server is running:

  • Swagger UI: http://localhost:8000/docs
  • ReDoc: http://localhost:8000/redoc

Example Usage

Get a specific verse

curl http://localhost:8000/api/v1/slok/1/1

Get random verse from chapter 2

curl http://localhost:8000/api/v1/slok/2

Get all chapters

curl http://localhost:8000/api/v1/chapters
